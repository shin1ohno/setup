# DVQ (Vision Quest) Template

A strategic document format inspired by Anthropic's DVQ practice: a 3-4 page written document that communicates vision, strategy, and direction to a team or organization.

## When to Use

- Quarterly or semi-annual strategic direction setting
- Major initiative kickoff requiring shared understanding
- Communicating a shift in priorities or approach

## Structure

### 1. Current State (0.5 page)

Where we are now. Use concrete metrics and facts, not adjectives.
- Key metrics that define the current position
- What has changed since the last strategic review
- External forces or constraints affecting us

### 2. Vision (0.5 page)

Where we want to be. A specific, measurable end state — not an aspirational slogan.
- The target state described concretely enough that someone could verify whether we reached it
- The timeframe for achieving this state
- Why this destination matters (the "so what")

### 3. Strategic Direction (1-1.5 pages)

How we get from current state to vision. The 2-4 key bets or shifts we are making.
- Each direction should be a choice that excludes alternatives (a true strategic decision, not a platitude)
- For each direction: what we are doing AND what we are explicitly not doing
- Dependencies and sequencing between directions

### 4. Concrete Actions (0.5-1 page)

The first 90 days of execution. Specific enough that someone could start working tomorrow.
- 3-5 actions with owners and deadlines
- Success criteria for each action
- How progress will be measured and communicated

### 5. Anticipated Questions (0.5 page)

Preempt the top 3-5 questions or objections the audience will have.
- Frame each as a genuine concern, not a strawman
- Provide a direct answer, not a deflection

## Writing Guidelines

- Total length: 3-4 pages (1500-2000 words)
- Written in first person plural ("we") to signal shared ownership
- Every claim backed by a number or a specific example
- No jargon without definition on first use
- Read aloud before finalizing — if a sentence requires re-reading, rewrite it
